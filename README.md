# E_Taxi
 Site de reservation de Taxi

dependances (vite.js)
- composer install
- npm install
- npm run build 

- php artisan migrate

- php db:seed --class=UsersSeeder
- php db:seed --class=ReservationsSeeder
- php artisan serve
